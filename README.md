# -Analyse-e-commerce-GCP-BigQuery-

Le projet vise à analyser et comparer la performance e-commerce entre 2023 et 2024 afin de produire des KPI fiables et un tableau de bord Power BI permettant d’éclairer les décisions business.

Ce projet consiste donc à réaliser une analyse de performance e-commerce pour l’entreprise TheLook Europe en tant que Data Analyst.

L'objectif est d’étudier l’activité commerciale sur un périmètre défini et de comparer les résultats entre 2023 et 2024 afin d’identifier les évolutions, tendances et leviers business.
